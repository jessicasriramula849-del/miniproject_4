# miniproject_4
Amazon Music Clustering

Brief summary of my project:
With millions of songs available on platforms like Amazon, manually categorizing tracks into genres is impractical. The goal of this project is to automatically group similar songs based on their audio characteristics using clustering techniques and exporting my analysis into a new dataset file and showcasing my results in streamlit application. By analyzing patterns in features such as tempo, energy, danceability, and more, my goal is to develop a model that organizes songs into meaningful clusters, potentially representing different musical genres or moods—without any prior labels. The main uses of this project are to automatically group songs that sound similar to enhance playlist generation, suggest similar tracks to users based on their preferred audio profile, help artists and producers identify competitive songs in the same audio cluster and streaming platforms can use clusters to analyze user listening patterns and optimize recommendations or promotions.
